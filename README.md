# ddu-source-buffer

buffer source for ddu.vim.

## Required

### denops.vim

https://github.com/vim-denops/denops.vim

### ddu.vim

https://github.com/Shougo/ddu.vim

## Configuration

```
" Use buffer source.
call ddu#start({'sources': [{'name': 'buffer'}]})
```
